This project is the complete rebuild of my personal developer portfolio, **LV-Portfolio**. The goal was to create a digital resume that not only showcases my projects and skills but also serves as a demonstration of my ability to deliver **high-quality, performant, and well-structured web applications**.

## âœ¨ Core Features & Design Engineering

The portfolio's architecture was deliberately engineered for speed and maintainability:

- **Performance Optimization (Next.js)**: Utilized Next.js features such as **Image Optimization**, **Static Site Generation (SSG)** for all static content, and intelligent code splitting to achieve near-perfect scores on Lighthouse metrics (e.g., Core Web Vitals).
- **Component-Driven UI**: Developed the entire interface using a **reusable component library** (e.g., `<ProjectCard>`, `<SkillBadge>`, `<TimelineItem>`). This modular approach ensures rapid styling changes and consistent UI/UX across all pages.
- **Modern Design System Implementation**: Established core design tokens (colors, typography, spacing) within a framework like **Styled-Components** or **Tailwind CSS**. This ensures every visual element adheres to a single source of truth.
- **Responsive and Accessible Design**: Implemented a **mobile-first** development strategy, ensuring the layout and content are fully responsive and meet standard **WCAG accessibility guidelines** (Aria labels, proper keyboard navigation) for broad reach.

## ðŸ’» Technologies & Stack

**Frontend Framework:** Next.js (App Router) - High-performance React framework for rendering and routing, enabling exceptional loading speeds and SEO optimization.
